# TaskFlow

A simple role-based web app built with React + Vite.

It provides separate **Admin** and **User** panels with authentication, protected routes and basic user management.

## Features

- Admin & User login
- Role based routing
- Protected routes
- My Profile (view & update)
- Member management (Add / Edit / Delete)
- Reusable UI components
- Toast based alerts & confirmations
- Clean responsive UI

## Tech Stack

- React (Vite)
- React Router
- Tailwind CSS
- Axios
- react-hot-toast

## Run locally

```bash
npm install
npm run dev
```
